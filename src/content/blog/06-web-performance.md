## บทนำ

Web performance เป็นปัจจัยสำคัญสำหรับ SEO และ user experience

## ทำไม Web Performance จึงสำคัญ?

- ผู้ใช้จะออกจากเว็บไซต์ที่โหลดช้า
- Google ใช้ page speed เป็น ranking factor
- Performance ส่งผลต่อ conversion rate

## Core Web Vitals

Google ใช้ Core Web Vitals เพื่อวัด page experience

### 1. Largest Contentful Paint (LCP)

เวลาที่ element ที่ใหญ่ที่สุดโหลดเสร็จ

### 2. First Input Delay (FID)

เวลาตอบสนองต่อการกระทำของผู้ใช้

### 3. Cumulative Layout Shift (CLS)

การเลื่อนตัวของ layout ขณะโหลด

## เทคนิคการปรับปรุง Performance

### Image Optimization

ใช้ modern formats และ lazy loading

### Code Splitting

ใช้ dynamic import

### Minification

ลด CSS, JavaScript, HTML

### Caching

ตั้งค่า browser caching

### CDN Usage

ใช้ Content Delivery Network

## ทดสอบ Performance

- Google PageSpeed Insights
- Lighthouse
- WebPageTest

## บทสรุป

Web performance optimization เป็นกระบวนการต่อเนื่อง
