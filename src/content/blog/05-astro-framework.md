## บทนำ

Astro เป็น framework ที่ใหม่และทรงพลังสำหรับสร้างเว็บไซต์ที่เร็ว

## Astro คืออะไร?

Astro เป็น static site generator ที่ช่วยให้คุณสร้างเว็บไซต์ที่:

- โหลดเร็วมาก
- SEO-friendly
- ใช้ JavaScript น้อยที่สุด

## ข้อดีของ Astro

### 1. Performance

Astro สร้าง static HTML ซึ่งโหลดได้เร็วมาก

### 2. SEO-Friendly

Static HTML ดีสำหรับ SEO

### 3. Flexibility

Astro รองรับ components จากหลาย frameworks

## การติดตั้ง Astro

```bash
npm create astro@latest my-project
cd my-project
npm install
npm run dev
```

## Deployment

Astro สามารถ deploy ได้ง่ายไปยังหลาย platforms

## บทสรุป

Astro เป็น framework ที่ยอดเยี่ยมสำหรับสร้างเว็บไซต์ที่เร็ว
